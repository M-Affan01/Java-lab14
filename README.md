# Lab 14 - GitHub Integration with Eclipse (Java Project)

## Instructor
Sir Usman Javed

## Objective
Create a GitHub account, clone repository in Eclipse, and push Java project to GitHub.

## Tools Required
- GitHub Account
- Eclipse IDE
- Java Project
- Internet Connection
- Git (optional)

## Steps

### 1. Create GitHub Account
- Go to https://github.com
- Sign up and verify email

### 2. Create Repository
- Click New Repository
- Name: Lab14-JavaProject
- Set visibility (Public/Private)
- Create repository

### 3. Clone Repository in Eclipse
- Open Eclipse
- File → Import → Git → Projects from Git
- Select Clone URI
- Paste repository URL
- Enter GitHub credentials/token
- Finish cloning

### 4. Add Java Project
- Import Java project into Eclipse
- Copy project into cloned repository folder
- Set Git repository using Team → Share Project

### 5. Commit Changes
- Right click project → Team → Commit
- Write message: Initial commit - Java project added
- Commit files

### 6. Push to GitHub
- Team → Push to Origin
- Select remote repository
- Push changes

## Result
Java project successfully uploaded to GitHub repository.

## Conclusion
Learned how to connect Eclipse with GitHub, clone repository, and push Java project using Git.
